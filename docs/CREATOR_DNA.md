✅ 一針見血

# 永遠不要做

- 無聊內容
- 為了流量而流量
- 空泛雞湯
- 沒有觀點

# 禁用語言（Banned Language）

以下語言和能量在所有內容中一律禁止。

**禁用詞語：**

- 一起成長
- 相信自己
- 拒絕是踏腳石
- 跟隨熱情
- 找到真正的夥伴
- 主動出擊
- 你會更有信心
- 勇敢踏出第一步
- 每一次都是學習
- 讓夢想起飛
- 讓我們一起
- 加油你可以的

**禁用句型：**

- 章節標題問句（失去朋友？）— 標題不是文案
- 表面 vs 真相（分類框架，不是觀點）
- 分享你的故事！（懶惰的 CTA）
- 成功典範（沒有思想的段落標題）
- 你有過這種經歷嗎？（lazy CTA）
- 一起成長，共創未來（空洞口號）
- 保險業的挑戰（描述類別，什麼都沒說）

# 聲音標準（Voice Standard）

每一張 slide 的 main_copy 必須包含：

**一個具體的逆轉或洞察**——點名保險業真實情境。不模糊，不勵志。

讀者反應必須是：「靠，這說的就是我」

**好的 main_copy 範例（用這個能量）：**

- 做保險不是讓你沒朋友，是你用錯方式開口。
- 朋友討厭的不是保險，是被推銷的感覺。
- 新人不是不適合，是一開始就走錯順序。
- 主管最危險的事，不是不教，而是教錯順序。
- 努力了三個月，業績還是零——問題不在你的個性。
- 你以為他拒絕你是因為不需要，其實他只是還沒信任你。
- 選主管比選公司更重要，這句話保險業很少人說清楚。

每一句都是完整想法，有具體逆轉，聽起來像真人寫的，不像 AI 模板。

# 品質門檻（Quality Threshold）

Phoenix Creative Director 採用 7 維度品質評分（1-10 分）：

| 維度 | 說明 |
|------|------|
| Topic Direction | 主題是否刺穿真實、具體的恐懼或洞察？ |
| Narrative Arc | 8 張 slide 邏輯是否清晰，沒有跳躍？ |
| Yusheng Voice | 粉絲會立刻認出是小佑的聲音嗎？ |
| Pain Precision | 痛點是否用業界專屬語言精準點名？ |
| Visual Executability | 所有場景是否真實、電影感、可執行？ |
| Publish Readiness | 今天可以直接發嗎？不需要任何修改？ |
| Overall | 以上的誠實平均分 |

**規則：** 所有單項分數 >= 9 才能標記 `publish_ready: true`。

任何一項低於 9 → 標記為「NEEDS REVISION」，列出原因和修改方向。

評分必須誠實。7 分暴露真問題 > 假的 10 分。

# 場景動態原則（Scene Motion Principle）

所有視覺背景必須是**真實電影人物場景**，絕對禁止抽象元素。

**禁止出現：**
光點流動 / 粒子漂浮 / 抽象能量 / 漸層呼吸

**好的 motion_background 範例：**
- 手機螢幕微微亮起，LINE 未回覆訊息停在畫面中央，背景咖啡廳人影慢速流動
- 夜晚辦公室只剩一盞桌燈，空白名單表被風吹得輕微翻動，時鐘秒針慢慢走
- 主管在白板前畫出三欄流程，白板筆跡逐步出現，背景保持暗色電影感
- 新人低頭看著手機，窗外車燈緩慢掠過，畫面有壓力但不混亂

每張 slide 的 animation_prompt 必須是給 motion designer 的電影級分鏡指示。
