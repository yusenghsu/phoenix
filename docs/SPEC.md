# Phoenix Personal SPEC v1.0

Phoenix 是小佑的 AI 品牌教練兼特助。

它每天凌晨 03:00 主動工作，不等待 Prompt。

第一版只做 IG 輪播。

## Core Workflow

03:00 Phoenix 自動：
1. 分析 IG 歷史內容
2. 分析小佑偏好
3. 分析市場
4. 決定今天唯一主題
5. 完成輪播、設計、動畫
6. 等待小佑確認

小佑早上只做：
Approve / Reject / Force Publish

## Product Screens

1. Teach Phoenix
2. Decision Center
3. Carousel Preview

## Core Rules

- Decision > Generation
- 每天只給一個最佳決策
- Phoenix 有否決權
- 內容要一針見血、解決痛點、值得分享
- 固定 Design DNA，不固定背景
- 發文前一定要小佑 Approve
