# Phoenix Personal V1 Demo Script

## Core Narrative

Phoenix is not a prompt-based content generator.

Phoenix is an AI brand coach that wakes up before the creator, makes the content decision, prepares the carousel, and waits for approval.

The creator's only job in the morning: review, confirm, publish.

---

## Demo Flow

### 1. Start at Today `/`

Open the homepage. Show the morning operating brief.

- "Good morning, 小佑." — Phoenix already finished the work at 03:00.
- **Today's Decision** — The topic Phoenix selected and its confidence score.
- **Today's Operating Brief** — Three status cards: Decision made, Carousel ready, Timing suggested.
- **Why today?** — One paragraph explaining the reasoning.
- **Ready to publish** — Four checklist items already complete.
- **Three buttons** — View Carousel, View Decision, Publish.

Key point: The creator doesn't have to think about what to post. Phoenix already decided.

---

### 2. Open Decision `/decision`

Click **View Decision** from the home page.

- **"Why I chose this."** — Phoenix explains the full decision rationale.
- **Confidence 92%** — Phoenix's conviction level on today's pick.
- **Decision Factors** — Four signal cards: Market Signal, Creator DNA Fit, Brand Memory, Share Worthiness.
- **Why today?** — Extended reasoning with a highlighted principle.
- **Rejected candidates** — Three topics Phoenix considered and rejected, with reasons.
- **Decision Matrix** — Score comparison across all four candidates.
- **Phoenix recommendation** — Three reasons for today's choice.
- **Risk if skipped** — What happens to brand memory if this topic is skipped.

Key point: Phoenix is transparent. Every decision has a full paper trail.

---

### 3. Open Carousel `/carousel`

Click **View Carousel** from the decision page or home page.

- **8-slide preview** — Navigate with arrow buttons or keyboard arrows.
- **Slide copy panel** — Shows the copy for each individual slide.
- **Caption** — Collapsed by default. Expand to see the full Instagram caption.
- **Copy Caption** — One click to copy. Toast confirmation appears.
- **Ready to publish checklist** — 5 items verified.
- **Export ready status** — Instagram 1:1 format, 8 slides.
- **Approve & Publish** — Confirms and moves to publish page.

Key point: The carousel draft is already done. The creator previews and approves.

---

### 4. Approve Publish `/publish`

Click **Approve & Publish** from the carousel page.

- **Published.** — Confirmation screen.
- **Scheduled status** — Instagram · Today 20:00.
- **Back Home** — Returns to the brief.

Key point: One tap to approve. Phoenix handles the scheduling.

---

### 5. Show Creator DNA `/settings`

Navigate via the header **Creator DNA** link.

- **Brand Voice** — Tags and description of how Phoenix understands the creator's tone.
- **Content Goal** — What "good content" means for this brand.
- **Content Formula** — The step-by-step content structure Phoenix uses.
- **Content Ratio** — Distribution across Viewpoint / Viral / Story content.
- **Phoenix will avoid** — List of content types Phoenix rejects.
- **Design DNA** — Visual brand references.
- **Decision Rules** — The five principles Phoenix follows for every decision.

Key point: This is Phoenix's memory. The more the creator teaches, the better Phoenix decides.

---

### 6. Show History `/history`

Navigate via the header **History** link.

- **7 decisions logged** — Phoenix has been working for a week.
- **Summary cards** — Best signal, strongest topic, rules learned.
- **Timeline** — Full record of each day: topic, decision, score, result.
- **Phoenix learned** — What Phoenix updated about its understanding after each post.
- **What Phoenix is learning** — Three evolving principles.

Key point: Phoenix learns from every decision. The system gets sharper over time.

---

### 7. Show Teach Phoenix `/teach`

Navigate via the header **Teach** link.

- **7-step onboarding wizard** — Teaches Phoenix the creator's brand from scratch.
- **Step 1** — Introduction to Phoenix's role.
- **Step 2** — Connect Instagram (mock).
- **Step 3** — Creator taste references.
- **Step 4** — Topic map.
- **Step 5** — Content avoidance list.
- **Step 6** — Brand asset upload (mock).
- **Step 7** — Creator DNA build animation.

Key point: This is the one-time setup that teaches Phoenix everything it needs to start making decisions.

---

## Current Prototype Limitations

- Mock data only — all content is pre-written
- No real Instagram connection
- No real OpenAI or AI model connection
- No database or storage
- No actual publishing or scheduling
- No carousel export to image files
- No real login or user accounts
